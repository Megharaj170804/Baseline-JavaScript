JavaScript DOM Navigation

DOM Nodes

Everything in an HTML document is a node: document node, element node, text node, attribute node (deprecated), and comment node.

<html>
  <head>
    <title>DOM Tutorial</title>
  </head>
  <body>
    <h1>DOM Lesson one</h1>
    <p>Hello world!</p>
  </body>
</html>

Node Relationships

  • Parent: The node above
  • Child: The node below
  • Sibling: Nodes with the same parent

Example: <h1> and <p> are siblings, both children of <body>.

Navigating Between Nodes

  • parentNode
  • childNodes[nodenumber]
  • firstChild
  • lastChild
  • nextSibling
  • previousSibling
