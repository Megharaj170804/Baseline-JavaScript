JavaScript Variables, Declaration, and Casting

Variable Declaration

Variables in JavaScript can be declared using var, let, or const:

  • var: Function-scoped, can be re-declared and updated.
  • let: Block-scoped, can be updated but not re-declared in the same scope.
  • const: Block-scoped, cannot be updated or re-declared. Must be initialized at declaration.

Type Casting (Conversion)

JavaScript allows you to convert (cast) data from one type to another:

  • String to Number: Number("123")
  • Number to String: String(123) or 123.toString()
  • Any to Boolean: Boolean(value)
  • Parse Int/Float: parseInt("123"), parseFloat("123.45")
