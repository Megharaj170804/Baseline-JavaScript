JavaScript Iterator Protocol

What is the Iterator Protocol?

The iterator protocol allows JavaScript objects to define or customize their iteration behavior, such as what values are looped over in a for...of loop.

An object is an iterator if it has a next() method that returns an object with value and done properties.

Built-in Iterables

Arrays, Strings, Maps, Sets, and more are built-in iterables in JavaScript.

for (let value of [1, 2, 3]) {
    console.log(value);
}

Custom Iterator Example

let myIterable = {
    [Symbol.iterator]: function() {
        let step = 0;
        return {
            next: function() {
                step++;
                if (step === 1) return { value: 'a', done: false };
                if (step === 2) return { value: 'b', done: false };
                return { value: undefined, done: true };
            }
        };
    }
};
for (let value of myIterable) {
    console.log(value);
}
