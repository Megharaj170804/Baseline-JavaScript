Mini Project: Form Validation System

This mini project demonstrates a complete user registration and sign-in system using HTML, CSS, Bootstrap, and JavaScript. It features:

  • Modern, responsive design with Bootstrap
  • Tabbed navigation for Sign In and Register
  • Client-side form validation with error and success messages
  • Local storage for user data (no backend required)
  • Secure password requirements
  • Instant feedback for registration and login attempts

After signing in, you are redirected to this page, which describes the project and its features. You can extend this project by adding user profile management, password reset, or connecting to a backend server.

How it works:

  1. Register a new user with name, email, and password.
  2. Sign in using your registered email and password.
  3. On successful sign in, you are redirected here.
