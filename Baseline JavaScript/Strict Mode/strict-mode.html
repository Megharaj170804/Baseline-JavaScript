JavaScript Strict Mode

What is Strict Mode?

Strict mode is a way to opt in to a restricted variant of JavaScript, making it easier to write secure and error-free code.

  • Prevents the use of undeclared variables
  • Throws errors for unsafe actions
  • Disables some problematic features

How to Enable Strict Mode

  • For scripts: Place "use strict"; at the top of your script file.
  • For functions: Place "use strict"; at the top of your function body.
  • For modules: Modules are always in strict mode by default.
  • For classes: Class bodies are always in strict mode by default.
