JavaScript Functions and Scope

Functions

A function is a block of code designed to perform a particular task. Functions can take parameters and return values.

function greet(name) {
    return "Hello, " + name;
}

Scope

Scope determines the accessibility of variables. JavaScript has function scope and block scope.

  • Global Scope: Variables declared outside any function are global.
  • Function Scope: Variables declared inside a function are only accessible within that function.
  • Block Scope: Variables declared with let or const inside a block ({}) are only accessible within that block.
