JavaScript Objects and Object Arrays

What is an Object?

An object in JavaScript is a collection of properties, where each property is a key-value pair.

let person = {
    name: "Alice",
    age: 30
};

Object Arrays

An object array is an array where each element is an object.

let people = [
    { name: "Bob", age: 25 },
    { name: "Carol", age: 28 }
];
