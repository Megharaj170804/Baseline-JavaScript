JavaScript Keywords and Identifiers

What are Keywords?

Keywords are reserved words in JavaScript that have special meaning. You cannot use them as variable names.

  • var
  • let
  • const
  • function
  • if
  • else
  • return

What are Identifiers?

Identifiers are names given to variables, functions, and objects. They must start with a letter, _, or $, and cannot be a keyword.

let name;
const $amount = 100;
function myFunction() { }
