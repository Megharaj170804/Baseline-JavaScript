JavaScript Control Statements: Loops

for Loop

The for loop repeats a block of code a specific number of times.

for (let i = 0; i < 5; i++) {
    // code to run
}

while Loop

The while loop repeats a block of code as long as a condition is true.

let i = 0;
while (i < 5) {
    // code to run
    i++;
}

forEach Loop

The forEach loop is used to execute a function for each element in an array.

let arr = [1, 2, 3];
arr.forEach(function(item) {
    // code to run for each item
});
