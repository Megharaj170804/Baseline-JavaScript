JavaScript DOM Collections and NodeLists

The HTMLCollection Object

getElementsByTagName() returns an HTMLCollection, which is an array-like list of HTML elements.

const myCollection = document.getElementsByTagName("p");

Access elements by index: myCollection[1]

Length property: myCollection.length

HTMLCollection is NOT an Array

You can loop through an HTMLCollection, but you cannot use array methods like pop(), push(), or join().

NodeList

querySelectorAll() returns a NodeList, which is also array-like and can be looped, but supports forEach().
