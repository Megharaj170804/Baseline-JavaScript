JavaScript Collections

What are Collections?

Collections in JavaScript are objects that group multiple values together. The most common collections are Array, Set, and Map.

Array

An Array is an ordered list of values. You can access elements by their index.

let arr = [1, 2, 3];

Set

A Set is a collection of unique values. No duplicates allowed.

let set = new Set([1, 2, 3, 2]); // Set {1, 2, 3}

Map

A Map is a collection of key-value pairs. Keys can be any type.

let map = new Map();
map.set('a', 1);
map.set('b', 2);
